# 🎬 Movie Recommendation System using Fuzzy AHP

An intelligent movie recommendation system that leverages **Fuzzy Analytic Hierarchy Process (Fuzzy AHP)** to provide personalized movie suggestions based on user preferences and multiple decision-making criteria.

The system combines decision science techniques with movie metadata analysis to help users discover films that best match their interests.

---

## 🚀 Features

### 🎯 Personalized Recommendations

* Multi-criteria movie evaluation
* Preference-based recommendations
* Intelligent ranking mechanism
* User-centric decision support

### 🧠 Fuzzy AHP Implementation

* Fuzzy pairwise comparison matrices
* Criteria weight calculation
* Consistency analysis
* Multi-Criteria Decision Making (MCDM)
